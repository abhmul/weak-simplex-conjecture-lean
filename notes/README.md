# Historical proof input

[`simplex_optimality_proof.v3.md`](simplex_optimality_proof.v3.md) is the mathematical proof draft used to design and implement the Lean formalization. It is retained because the work-package records cite its sections and because it documents the correspondence between the informal argument and the code.

The authoritative paper is Abhijeet Mulgund, [*Stochastic Domination of Gaussian Maxima: A Resolution of the Weak Simplex Conjecture*](https://arxiv.org/abs/2607.14087), arXiv:2607.14087. The historical draft in this directory predates the arXiv manuscript and should not be cited in place of the paper.

The formalization architecture and trust-boundary report is maintained at [`docs/weak_simplex_lean_formalization_report.v3-reassessment.md`](../docs/weak_simplex_lean_formalization_report.v3-reassessment.md).
