### C. Positive quadratic coercivity

Search once more after WP02 exposes its wrappers, then prototype the report's exact local theorem:

```lean
lemma Matrix.PosDef.exists_quadratic_lower_bound
    {m : ℕ} {A : Matrix (Fin m) (Fin m) ℝ}
    (hA : A.PosDef) :
    ∃ κ : ℝ, 0 < κ ∧
      ∀ x : WeakSimplex.Coord m,
        κ * ‖x‖ ^ 2 ≤ ⟪x, Matrix.toEuclideanCLM A x⟫_ℝ
```

Use compactness of the unit sphere and strict positivity unless an exact pinned theorem is found. Keep this in `Scratch/` until the WP02 API compiles and the integration owner assigns its production home.

The key pinned candidates are `Real.map_matrix_volume_pi_eq_smul_volume_pi`, `Matrix.det_sqrt`, `Matrix.PosDef.det_pos`, `Measure.withDensity_mul`, and `withDensity_smul_measure`.
