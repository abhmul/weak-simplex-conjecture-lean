## Mathematical role

For `m > 0`, define the coordinate maximum of `x : Coord m` by the nonempty finite supremum

```text
coordinateMax hm x = max i, x i.
```

Its sublevel set is exactly `lowerOrthant c`. WP19 says the corresponding CDF under every
admissible covariance `R` is at least the independent standard-Gaussian CDF. Thus the upper tails
of the maximum under `R` are no larger. Applying this tail order to the nonnegative increasing map
`x ↦ exp (μ x)` gives the exponential-moment comparison for `μ ≥ 0`.

The right-hand Bochner integral is genuine: for `μ ≥ 0`,

```text
exp (μ * max i, x i) ≤ ∑ i, exp (μ * x i),
```

and every coordinate of the identity-covariance Gaussian has the standard one-dimensional
Gaussian law and finite MGF. This package closes only the maxima/MGF layer. Codebook covariance
normalization belongs to WP21 and the Bayes/ML value identity belongs to WP22.

Relevant plan sections are §§2.2–2.3, 6.2–6.3, 10, 11.7, 12 (WP20), 13, and 18. The relevant
mathematical source sections are §§2.5 and 10 of `notes/simplex_optimality_proof.v3.md`.

## Frozen public API

The production modules expose exactly the following declarations:

```lean
namespace WeakSimplex

/-- The largest coordinate of a nonempty Euclidean coordinate vector. -/
def coordinateMax {m : ℕ} (hm : 0 < m) (x : Coord m) : ℝ

/-- The coordinate maximum is continuous. -/
theorem continuous_coordinateMax {m : ℕ} (hm : 0 < m) :
    Continuous (coordinateMax hm)

/-- A coordinate maximum is at most `c` exactly on the lower orthant at `c`. -/
theorem coordinateMax_le_iff_mem_lowerOrthant
    {m : ℕ} (hm : 0 < m) (x : Coord m) (c : ℝ) :
    coordinateMax hm x ≤ c ↔ x ∈ lowerOrthant c

/-- WP19 expressed as upper-tail domination for coordinate maxima. -/
theorem coordinateMax_tail_le_iid
    {m : ℕ} (hm : 0 < m)
    (R : Matrix (Fin m) (Fin m) ℝ)
    (hR : IsWeakSimplexCov R)
    (c : ℝ) :
    (multivariateGaussian (0 : Coord m) R) {x | c < coordinateMax hm x} ≤
      (multivariateGaussian (0 : Coord m) (1 : Matrix (Fin m) (Fin m) ℝ))
        {x | c < coordinateMax hm x}

/-- Admissible Gaussian coordinate maxima have no larger nonnegative exponential moments than
the independent standard-Gaussian maximum. -/
theorem gaussianMax_mgf_le_regularSimplex
    {m : ℕ} (hm : 0 < m)
    (R : Matrix (Fin m) (Fin m) ℝ)
    (hR : IsWeakSimplexCov R)
    (μ : ℝ) (hμ : 0 ≤ μ) :
    mgf (coordinateMax hm) (multivariateGaussian 0 R) μ ≤
      mgf (coordinateMax hm)
        (multivariateGaussian 0 (1 : Matrix (Fin m) (Fin m) ℝ)) μ

end WeakSimplex
```

All other definitions and theorems are private. The acceptance theorem uses the real-valued
mathlib `mgf`. Its proof establishes integrability before comparing Bochner integrals; it does not
exploit the convention that a nonintegrable Bochner integral is zero.

- Do not assume a stochastic-order theorem, MGF finiteness, the identity-covariance orthant
  formula, or coordinate-max measurability as a production parameter.
- Do not define a measurable `argmax`; only the pointwise finite maximum is needed.
- Do not replace the maximum with `max 0`, exclude negative thresholds, or strengthen
  `IsWeakSimplexCov`.
- Do not state the result only for positive-definite `R`; WP19 must be used at this boundary.
- Do not use the Bochner-integral-zero convention for nonintegrable functions.
