## Mathematical role

Create one stable real-valued standard-normal PDF/CDF interface, bridge it once to the measure-theoretic CDF, and prove the two truncated moments used by the adaptive and product branches.

Relevant plan sections: §§5, 6.1, 11.2, 12 (WP03), 13, 14, and 17 (days 6–10). Relevant mathematical source: §§2.2, 7, and 7.1 of `notes/simplex_optimality_proof.v3.md`.

## Frozen public API

```lean
namespace WeakSimplex

noncomputable section

open MeasureTheory ProbabilityTheory
open scoped ENNReal

def normalPDF (x : ℝ) : ℝ := gaussianPDFReal 0 1 x

def normalCDF (x : ℝ) : ℝ := ∫ t in Set.Iic x, normalPDF t

lemma normalPDF_pos (x : ℝ) : 0 < normalPDF x

lemma hasDerivAt_normalPDF (x : ℝ) :
    HasDerivAt normalPDF (-x * normalPDF x) x

lemma normalCDF_eq_measure_Iic (x : ℝ) :
    ENNReal.ofReal (normalCDF x) = (gaussianReal 0 1) (Set.Iic x)

lemma normalCDF_pos (x : ℝ) : 0 < normalCDF x

lemma normalCDF_lt_one (x : ℝ) : normalCDF x < 1

lemma hasDerivAt_normalCDF (x : ℝ) :
    HasDerivAt normalCDF (normalPDF x) x

lemma tendsto_normalCDF_atTop : Tendsto normalCDF atTop (𝒩 1)

lemma tendsto_normalCDF_atBot : Tendsto normalCDF atBot (𝒩 0)

lemma truncated_first_moment (s : ℝ) :
    ∫ x in Set.Iic s, x * normalPDF x = -normalPDF s

lemma truncated_second_moment (s : ℝ) :
    ∫ x in Set.Iic s, x ^ 2 * normalPDF x = normalCDF s - s * normalPDF s

end

end WeakSimplex
```

Helper lemmas such as PDF/CDF nonnegativity, `normalCDF_eq_cdf`, moment integrability, or Gaussian boundary-product limits may remain private unless their reuse is demonstrated. Do not change the public theorem headers after proof work begins.

## Implementation log

- No packaged derivative theorem for the `Set.Iic` primitive was found. The compiled proof rewrites differences of `Iic` integrals with `intervalIntegral.integral_Iic_sub_Iic` and applies `intervalIntegral.integral_hasDerivAt_right`.
- Strict CDF positivity and the strict upper bound use full support of the positive Gaussian density, positive `Iic`/`Ioi` volume, and the total-density integral rather than a new Gaussian-tail theorem.
- The endpoint limits pass through one private equality with `ProbabilityTheory.cdf`; the public definition remains the frozen real set integral and the `ENNReal` measure bridge has the required direction.
- First- and second-moment integrability are transported from `memLp_id_gaussianReal` through `integrable_withDensity_iff`; no polynomial-decay library was introduced.
- The two needed boundary products tend to zero via `tendsto_zero_of_hasDerivAt_of_integrableOn_Iic` and the already established derivative/integrability facts.
- A prototype initially exposed a function-negation extensionality mismatch. Choosing the derivative function pointwise as `fun x ↦ -(x * normalPDF x)` preserved the theorem header and compiled directly.
