## Mathematical role

For a unit codebook `code : Fin m → Coord n`, let the likelihood ratio of class `i` relative to
standard Gaussian noise be

```text
L_i(y) = exp (lam * ⟪code i, y⟫ - lam^2 * ‖code i‖^2 / 2).
```

The uniform-prior Bayes value is `1/m` times the standard-Gaussian integral of the pointwise finite
maximum of the `L_i`. Under the unit-norm hypothesis, this maximum is

```text
exp (-lam^2 / 2) * exp (lam * max_i ⟪code i, y⟫).
```

The Gaussian score vector has covariance equal to the code Gram matrix, so the Bayes value equals
the prefactored MGF from WP21. The code Gram matrix is a correlation matrix. WP21 therefore bounds
the value by the regular-simplex matrix target. The final theorem specializes to `m = n + 1`.

The Bayes value must be defined through the pointwise finite maximum before introducing a decoder.
A deterministic measurable maximizing selector may then certify the operational ML statement and
make duplicate codewords and ties explicit. The analytic and normalization layers remain untouched.

Relevant plan sections are §§2.2–2.3, 11.7, 12 (WP22), 13, 15, and 18. The relevant mathematical
source is §3 of `notes/simplex_optimality_proof.v3.md`.

## Frozen acceptance API

The single acceptance theorem is:

```lean
namespace WeakSimplex

theorem weak_simplex
    {n : ℕ} (hn : 0 < n)
    (code simplex : Fin (n + 1) → Coord n)
    (hunit : ∀ i, ‖code i‖ = 1)
    (hsimplex : codeGram simplex = regularSimplexGram (n + 1))
    (lam : ℝ) (hlam : 0 ≤ lam) :
    decoderSuccess (Nat.succ_pos n) code lam ≤
      decoderSuccess (Nat.succ_pos n) simplex lam

end WeakSimplex
```

This statement was frozen only after the score-law, Bayes-integral, selector, shifted-Gaussian
success identity, and final combined prototypes compiled through actual imports. The supplied
regular-simplex realization is identified by its exact Gram matrix; its unit norms are derived.
The theorem makes no distinct-codeword or null-tie assumption. The nonnegative endpoint
`lam = 0` is valid and strengthens the source's positive-signal statement.
