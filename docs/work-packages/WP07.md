## Mathematical role

Differentiate the unconstrained adaptive potential only along one coordinate at a time. Global
maximality then forces the stationarity equation, which yields the change-of-measure compatibility
relation and rewrites the maximizer value in the exact form consumed by WP09.

Relevant plan sections: §§3.3–3.6, 5.3, 11.3, 12 (WP07), 13, and 17. Relevant mathematical
source: §§8.1–8.2 of `notes/simplex_optimality_proof.v3.md`, with the reassessment's unconstrained
`s`-space parameterization taking precedence over the source's endpoint parameterization.

namespace WeakSimplex

def coordinateLine {m : ℕ}
    (s : Coord m) (i : Fin m) (t : ℝ) : Coord m :=
  Coord.ofFun fun j ↦ s j + t * if j = i then 1 else 0

@[simp]
lemma coordinateLine_apply {m : ℕ}
    (s : Coord m) (i j : Fin m) (t : ℝ) :
    coordinateLine s i t j = s j + t * if j = i then 1 else 0

@[simp]
lemma coordinateLine_zero {m : ℕ} (s : Coord m) (i : Fin m) :
    coordinateLine s i 0 = s

def adaptivePotentialCoordinateLine {m : ℕ}
    (R : Matrix (Fin m) (Fin m) ℝ) (c : ℝ)
    (s : Coord m) (i : Fin m) : ℝ → ℝ :=
  fun t ↦ adaptivePotential R c (coordinateLine s i t)

lemma hasDerivAt_adaptivePotentialCoordinateLine
    {m : ℕ} {R : Matrix (Fin m) (Fin m) ℝ} (hR : R.PosDef)
    (c : ℝ) (s : Coord m) (i : Fin m) :
    HasDerivAt (adaptivePotentialCoordinateLine R c s i)
      ((1 - s i * r (s i) - (r (s i)) ^ 2) *
        (r (s i) - matrixMul R⁻¹ (displacement c s) i)) 0

lemma adaptivePotential_stationary_coordinate
    {m : ℕ} {R : Matrix (Fin m) (Fin m) ℝ} (hR : R.PosDef)
    (c : ℝ) (s : Coord m)
    (hmax : ∀ t : Coord m,
      adaptivePotential R c t ≤ adaptivePotential R c s)
    (i : Fin m) :
    r (s i) = matrixMul R⁻¹ (displacement c s) i

lemma coordinateMap_r_eq_inv_displacement
    {m : ℕ} {R : Matrix (Fin m) (Fin m) ℝ} (hR : R.PosDef)
    (c : ℝ) (s : Coord m)
    (hmax : ∀ t : Coord m,
      adaptivePotential R c t ≤ adaptivePotential R c s) :
    coordinateMap r s = matrixMul R⁻¹ (displacement c s)

lemma adaptivePotential_compatibility
    {m : ℕ} {R : Matrix (Fin m) (Fin m) ℝ} (hR : R.PosDef)
    (c : ℝ) (s : Coord m)
    (hmax : ∀ t : Coord m,
      adaptivePotential R c t ≤ adaptivePotential R c s) :
    s + coordinateMap r s - matrixMul R (coordinateMap r s) =
      c • allOnesVector m

lemma adaptivePotential_value_identity
    {m : ℕ} {R : Matrix (Fin m) (Fin m) ℝ} (hR : R.PosDef)
    (c : ℝ) (s : Coord m)
    (hmax : ∀ t : Coord m,
      adaptivePotential R c t ≤ adaptivePotential R c s) :

      (∑ i, Real.log (normalCDF (s i))) +
        (⟪coordinateMap r s, coordinateMap r s⟫_ℝ -
          ⟪coordinateMap r s, matrixMul R (coordinateMap r s)⟫_ℝ) / 2

end WeakSimplex

namespace WeakSimplex

structure AdaptiveWitnesses {m : ℕ}
    (R : Matrix (Fin m) (Fin m) ℝ) (c : ℝ) where
  s : Coord m
  a : Coord m
  a_eq_r : ∀ i, a i = r (s i)
  a_pos : ∀ i, 0 < a i
  compatibility : s + a - matrixMul R a = c • allOnesVector m
  value_bound :
    (∑ i, Real.log (normalCDF (s i))) +
        (⟪a, a⟫_ℝ - ⟪a, matrixMul R a⟫_ℝ) / 2 ≥
      (m : ℝ) * Real.log (normalCDF c)

## Proof architecture

1. Keep the scalar coordinate-component, matrix-row, and qform-line helpers private. Prove the
   public derivative by definitional equality with `adaptivePotentialCoordinateLine`.
2. Convert global maximality to `IsLocalMax` at zero, apply
   `IsLocalMax.hasDerivAt_eq_zero`, and divide by the strictly positive `H'` factor using
   `H_deriv_pos`.
3. Use coordinate extensionality to obtain `coordinateMap r s = R⁻¹ displacement`, then
   `matrixMul_inv_right` to obtain the compatibility relation.
4. Rewrite the quadratic cost using the same inverse-displacement identity and real-inner-product
   symmetry. Rewrite the local square sum using `PiLp.inner_apply`.
5. Apply the WP06 maximizer-with-value theorem and package `a := coordinateMap r s`.

## Implementation log

- Direct PiLp-valued differentiation produced an instance diamond, so the production proof follows
  the compiled scalar finite-sum route. Coordinate components, displacement components, matrix
  rows, qform terms, and the qform line are all private.
- The quadratic derivative expands both product-rule terms. Symmetry of `R⁻¹` identifies them,
  giving the factor `2 * H' * (R⁻¹ w)_i`; the potential's `-1 / 2` coefficient then yields the
  frozen derivative exactly.
- Global maximality restricts to an `IsLocalMax` at zero. The derivative-zero theorem and
  `H_deriv_pos` force the coordinate residual to vanish without a division operation.
- Vector stationarity uses coordinate extensionality. `matrixMul_inv_right` has the required
  direction from `a = R⁻¹ w` to `R a = w`; coordinate simplification then proves compatibility.
- The value proof rewrites `qform R⁻¹ w` as `⟪a, R a⟫_ℝ`, rewrites the local reciprocal-Mills
  squares with `PiLp.inner_apply`, and closes the scalar half-factor algebra by `ring`.
- The card's cold-build gate order records that direct Lean checking does not install a new project
  object file before the dependent module is checked.
