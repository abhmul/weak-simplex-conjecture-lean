## Mathematical role

This package closes the adaptive branch at the conditional M1 boundary. It proves the report's

product branch will prove that property in WP17; the integration package will then expose the frozen

that final name with a temporary dependency parameter.

## Frozen public API

`Product/CenteredProperty.lean`:

```lean
namespace WeakSimplex

def CenteredProductProperty {m : ℕ}
    (R : Matrix (Fin m) (Fin m) ℝ) : Prop :=
  ∀ (f : Fin m → ℝ → ℝ),
    (∀ i, Measurable (f i)) →
    (∀ i x, 0 ≤ f i x) →
    (∀ i, Bornology.IsBounded (Set.range (f i))) →
    (∀ i, IsLogConcave (fun x ↦ ENNReal.ofReal (f i x))) →
    (∀ i, 0 < ∫ x, f i x ∂gaussianReal 0 1) →
    (∀ i, ∫ x, x * f i x ∂gaussianReal 0 1 = 0) →
    (∏ i, ∫ x, f i x ∂gaussianReal 0 1) ≤
      ∫ x, ∏ i, f i (x i) ∂multivariateGaussian 0 R

end WeakSimplex
```

The explicit measurability hypothesis is mathematically substantive, not a compiler workaround.
The project's `IsLogConcave` is intentionally a pointwise ENNReal inequality with no measurability
field; without this hypothesis the real Bochner integrals admit nonmeasurable factors and formal
integral-zero behavior. Do not add separate integrability or nonzero hypotheses: bounded measurable
factors are integrable under Gaussian probability measures, and positive mass already excludes the
zero factor.

## Proof architecture

1. Obtain `AdaptiveWitnesses R c` and instantiate the centered-product property with
   `centeredTiltedHalfLine (w.s i)`.
2. Prove the finite adaptive product measurable, nonnegative, globally bounded, and integrable.
3. Identify it pointwise with `exp ⟪w.a,x⟫` times the adaptive endpoint indicator.
4. Convert its real integral to an ENNReal lintegral and apply WP08's Gaussian shift and event
   preimage identity.
5. Rewrite every one-dimensional mass and exponentiate `w.value_bound`.
6. Convert `normalCDF c` back to the standard Gaussian lower-half-line measure.
