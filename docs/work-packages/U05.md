## Frozen public API

```lean
lemma H_pos (s : ℝ) : 0 < H s

def normalizedCenteredTiltedHalfLine (s x : ℝ) : ℝ :=
  (∫ z, centeredTiltedHalfLine s z ∂gaussianReal 0 1)⁻¹ *
    centeredTiltedHalfLine s x

theorem continuous_normalizedSelfConvolution_normalizedCenteredTiltedHalfLine
    (s : ℝ) :
    Continuous
      (normalizedSelfConvolution (normalizedCenteredTiltedHalfLine s))
```

Also expose conventionally named normalization lemmas for measurability, nonnegativity, bounded range, log-concavity, mass one, and zero Gaussian barycenter. Do not expose proof-specific piecewise algebra unless a downstream package needs it.

## Method

Prove `H_pos` from the existing Mills and derivative API. Normalize using the already proved positive mass of `centeredTiltedHalfLine`. Expand the self-convolution specialized to the truncated exponential factor, express its interval mass with `normalCDF`, and prove the pieces meet at `u = Real.sqrt 2 * H s`.
