## Mathematical role

WP22 constructed one measurable least-index maximum-likelihood decoder and proved that its average success equals the Bayes value. The mathematical source states the operational theorem for any measurable tie-breaking rule that always selects a maximizing score. This package exposes the generic decision-rule success value, proves that every measurable likelihood-maximizing decoder has the same Bayes value, and derives the weak-simplex comparison for arbitrary measurable score-maximizing decoders on both codebooks.

This is an explicit completion of the operational layer already deferred to WP22 by the governing plan. It does not alter the analytic, Gram, or supplied regular-simplex realization interfaces.

Relevant plan sections are §§1.1, 2.2, 12 (WP22), 15, and 18 (M4). The relevant mathematical source is §3 of `notes/simplex_optimality_proof.v3.md`.

## Frozen acceptance API

The single acceptance theorem is:

```lean
namespace WeakSimplex

theorem weak_simplex_of_scoreMaximizingDecoders
    {n : ℕ} (hn : 0 < n)
    (code simplex : Fin (n + 1) → Coord n)
    (hunit : ∀ i, ‖code i‖ = 1)
    (hsimplex : codeGram simplex = regularSimplexGram (n + 1))
    (lam : ℝ) (hlam : 0 ≤ lam)
    (decoder simplexDecoder : Coord n → Fin (n + 1))
    (hdecoder_meas : Measurable decoder)
    (hdecoder_max : IsScoreMaximizingDecoder code decoder)
    (hsimplexDecoder_meas : Measurable simplexDecoder)
    (hsimplexDecoder_max : IsScoreMaximizingDecoder simplex simplexDecoder) :
    decoderSuccessOf code lam decoder ≤
      decoderSuccessOf simplex lam simplexDecoder

end WeakSimplex
```

The supplied regular-simplex realization remains identified by its exact Gram matrix. This package does not add a canonical realization theorem; that omission is an intentional architecture choice, not a tie-breaking assumption.

## Frozen supporting surface

- `IsLikelihoodMaximizingDecoder`, the pointwise likelihood-maximizer predicate;
- `IsScoreMaximizingDecoder`, the equal-energy score-maximizer predicate used by the source;
- `decoderSuccessOf`, the uniform-prior operational success of an arbitrary decision rule;
- score maximization implies likelihood maximization for unit codewords and `lam ≥ 0`;
- `decoderSuccessOf_eq_bayesValue` for every measurable likelihood-maximizing decoder;
- equality of the success values of any two measurable likelihood-maximizing decoders;
- the existing `decoderSuccess` and `weak_simplex` signatures remain unchanged.

Generic finite-selector and indicator-partition machinery remains private.
