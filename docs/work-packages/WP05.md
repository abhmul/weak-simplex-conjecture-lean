## Mathematical role

Supply the positive-definite quadratic coercivity and inverse identities used by the compact-superlevel argument, then prove the rank-one inverse bound directly from `R - J/m ⪰ 0` without matrix square roots.

## Proof architecture

- Coercivity splits the empty-dimensional case, then minimizes `qform A` on the nonempty compact unit sphere and rescales an arbitrary vector. Do not introduce spectral eigenvalues unless the compactness route meets the stop condition.
- For the acceptance theorem, set `x = R⁻¹ 1` and `t = 1ᵀx`. Evaluating `R - J/m` at `x` gives `0 ≤ t - t²/m`; `hR.inv` and `allOnesVector m ≠ 0` give `0 < t`; ordered-field algebra yields `t ≤ m`.
- The explicit `hm` is mandatory. Lean's `1 / (0 : ℝ) = 0` makes the predicates type-correct at `m = 0`, but the rank-one normalization is mathematically dimension-positive.

## Implementation log

- The coercivity proof uses the compact unit sphere exactly as frozen. It handles `m = 0` by subsingleton elimination, constructs an explicit unit vector when `0 < m`, minimizes the continuous quadratic form, proves the minimum positive, and rescales arbitrary nonzero vectors.
- WP01's scratch theorem confirmed the compact-sphere route first. The production proof was adapted to the project API and narrow card imports; it does not retain the scratch prototype's unnecessary Rayleigh import.
- Bilinear self-adjointness uses `Matrix.inner_toEuclideanCLM`, `Matrix.dotProduct_transpose_mulVec`, and positive-definite symmetry. The diagonal-only `qform_eq_dotProduct` bridge is used only for quadratic positivity.
- Inverse action uses `map_mul` for `Matrix.toEuclideanCLM`, followed by `congrArg (fun T ↦ T x)`; no new inverse-positive-definite wrapper was added.
- The rank-one proof required only private expansions for matrix subtraction/scaling, the all-ones quadratic form, and the inner product with the all-ones vector. Scratch attempt one reduced the semidefinite hypothesis to `0 ≤ t - t ^ 2 / m`.
- Multiplying by positive `m`, factoring as `t * (m - t)`, and using `t > 0` closed the acceptance inequality. No matrix square root, eigenvalue, or spectral theorem appears in the rank-one module.
